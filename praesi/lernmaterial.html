Klausur-Spickzettel — das Wichtigste pro System

Roter Faden: vom Steuergestänge zum Avionik-Netzwerk — jedes System bringt mehr Automatik und setzt konsequent auf Redundanz.

01 · FADEC

Triebwerk

  • Full Authority: keine Mechanik Schubhebel↔FMU; EEC = Rechner, FADEC = Gesamtsystem
  • Pilot gibt nur den Sollwert — regelt Treibstoff/Drehzahl/Grenzwerte selbst
  • Regelgröße EPR / N1; Envelope Protection (Stalls · EGT · Hot Starts)
  • Redundanz: Dual-Channel A/B (fail-operational) + PMG → autark; ETOPS-Backup
02 · Fly-by-Wire

Steuerung

  • Signal statt Seil; Pilot gibt Steuerwunsch, FCC + SAS rechnen/dämpfen (Phugoide)
  • Irreversibel → Artificial Feel; kompakte Inceptors → freie Display-Sicht
  • Architektur: IMA (ARINC 653) + AFDX; Power-by-Wire (EHA/EMA)
  • Redundanz Duo-Duplex / Triple-Triple; Totalausfall Catastrophic 10⁻⁹/h
03 · Autopilot & FMS

Führung

  • FMS = Stratege (4D: Ort + Zeit), AP = Taktiker → Nick-/Rollwinkel für FBW
  • Kaskadierte Schleifen: außen Bahn (langsam), innen Lage (schnell), Ist-Soll → 0
  • Kalman (GPS + IRS, Schuler-Drift); LNAV/VNAV · Cost Index · Top of Descent
  • Hard (Airbus) vs Soft Limits (Boeing); Autoland CAT IIIb
04 · Autonomie

KI & Risiken

  • ALFUS (Mission · Umgebung · Human Independence); hoch-automatisiert ≠ autonom
  • Chancen: Auto-GCAS (Hypoxie/CFIT), FORDEC (KI-Co-Pilot), Risiko < 10⁻⁵
  • Risiken: Skill Degradation + Modus-Konfusion (Direct Law, AF447)
  • Black-Box/DO-178C → EASA Learning + Runtime Assurance
Kernsatz: Sicherheit = gute Technik + fähiger Mensch
Lernmaterial · Glossar

Fachbegriffe kompakt

FADEC / EEC

Full Authority Digital Engine Control = Gesamtsystem; EEC = der Rechner. Voll elektronische Triebwerksregelung ohne Mechanik.

EPR · N1

Regelgrößen für den Schub: EPR = Druckverhältnis (Auslass/Einlass), N1 = Fandrehzahl.

PMG

Permanentmagnet-Generator: versorgt die EEC ab Mindestdrehzahl unabhängig vom Bordnetz.

Fly-by-Wire

Eingaben werden elektrische Signale; Rechner (FCC) setzen sie um. Irreversibel → künstliches Steuergefühl.

FCC · SAS

Flight Control Computer übersetzt in Zielvorgaben; Stability Augmentation System dämpft Eigendynamik (z. B. Phugoide).

Artificial Feel

Federpaket erzeugt den Steuerdruck künstlich, da kein mechanisches Wind-Feedback existiert.

IMA · AFDX

Integrierte Modulare Avionik (ARINC 653) auf Zentralrechnern; AFDX/ARINC 664 = deterministisches Ethernet.

Power-by-Wire

EHA/EMA erzeugen die Stellkraft lokal aus Strom statt aus Zentralhydraulik (More Electric Aircraft).

FMS · Autopilot

FMS = Stratege (4D-Route), Autopilot = Taktiker (kaskadierte Regelschleifen: außen Bahn, innen Lage).

Kalman-Filter

Fusioniert GPS + IRS zu einer präzisen Position; GPS korrigiert die IRS-Drift (Schuler).

Cost Index · ToD

CI = Zeit- ÷ Treibstoffkosten; Top of Descent = optimaler Sinkflugpunkt (Continuous Descent).

Hard vs. Soft Limits

Airbus verweigert gefährliche Eingaben (hart); Boeing warnt, ist aber überstimmbar (weich).

CAT IIIb

Automatische Landung bei nahezu null Sicht (Flugzeug + Crew + Flughafen zertifiziert).

ALFUS

Autonomie-Modell mit drei Achsen: Mission-Komplexität, Umgebung, Human Independence.

Auto-GCAS · FORDEC

Auto-GCAS = autonomer Bodenkollisionsschutz; FORDEC = KI-Entscheidungszyklus.

Modus-Konfusion

Abrupte Kontrollrückgabe (z. B. „Direct Law") überfordert die Crew (Bsp. AF447).

DO-178C

Zulassungsnorm mit deterministischer Nachvollziehbarkeit — ungeeignet für „Black-Box"-KI.

Learning / Runtime Assurance

EASA-Ansatz: KI am Boden geprüft & „eingefroren" + deterministischer Wächter im Flug.
